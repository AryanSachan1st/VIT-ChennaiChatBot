Frontend Components of VIT Chennai AI Chatbot Application

The frontend of the VIT Chennai AI Chatbot consists of several HTML, CSS, and JavaScript files that create the user interface for interacting with the AI assistant.

1. **chatbot.html**
   - Main chat interface page
   - Contains the chat container with header, messages area, and input section
   - Header displays the title "VIT Chennai Chatbot" and description
   - Messages area shows conversation between user and bot
   - Input section has a text field and send button
   - Logout button for user authentication management
   - Links to styles.css for styling and script.js for functionality

2. **styles.css**
   - Defines the visual appearance of the chatbot interface
   - Uses VIT Chennai's official colors (#1a2a6c and #b21f1f) in gradients
   - Responsive design that works on different screen sizes
   - Styling for chat container, header, messages, and input area
   - Message bubbles with different styles for user and bot messages
   - Animations for message appearance and typing indicators
   - Hover effects for buttons to improve user experience

3. **script.js**
   - Handles client-side interactivity of the chatbot
   - Manages user input through text field and send button
   - Implements real-time chat functionality
   - Communicates with backend server via fetch API
   - Processes and displays responses from the AI assistant
   - Handles user authentication (login/logout)
   - Manages session state and user experience flow
   - Includes error handling for network requests and user interactions

User Experience Features:
- Clean, modern interface with college branding
- Real-time conversation flow
- Visual feedback for user actions
- Responsive design for mobile and desktop
- Typing indicators to show when bot is "thinking"
- Timestamps on messages
- Smooth animations for message transitions
- Intuitive input methods (button click or Enter key)

The frontend is designed to be user-friendly while maintaining the professional appearance expected for a college application. It provides a seamless interface for students and visitors to interact with the AI assistant and get information about VIT Chennai.
